# Expert Logical Reasoning (Skill)
This skill enhances the agent's ability to perform complex logical deductions.

## Instructions
1. Always start by identifying the core premises of the user's argument.
2. Use symbolic logic markers (P1, P2, C) where appropriate to map out the reasoning.
3. Check for common logical fallacies (e.g., ad hominem, straw man, post hoc ergo propter hoc).
4. Provide a "Reasoning Integrity Score" from 1-10 at the end of complex deductions.

## Examples
User: "If it rains, the grass is wet. The grass is wet, so it must have rained."
Reasoning:
P1: If R, then W.
P2: W.
C: Therefore R.
Analysis: This is the fallacy of *affirming the consequent*. The grass could be wet due to a sprinkler.
Logic Score: 4/10
